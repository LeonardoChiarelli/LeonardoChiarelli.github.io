Modelo de caixas

  • height - altura
  • width - largura
  • border - borda (linha em volta do conteúdo)
    • padding - acolchoamento ou preenchimento (espaço interno do 'border')
    • margin - margem (espaço externo do 'border')
      • outline - contorno ou traçado (dentro da margin)
  • Shorthand
    • border
      • Ordem
        1. width
        2. style
        3. color
    • padding
      • Ordem
        1. top
        2. right
        3. bottom
        4. left
    • margin
      • Ordem
        1. top
        2. right
        3. bottom
        4. left
    • outline
      • Ordem
        1. width
        2. style
        3. color

Tipos de caixa

  • box-level
    • Características
      • Sempre inicia em uma linha nova
      • Sempre ocupa a largura total da tela
      • Sempre pula para a próxima linha para continuar o conteúdo
    • Exemplos de tag's
      • <div>
      • <h1>-<h6>
      • <p>
      • <main>
      • <header>
      • <nav>
      • <article>
      • <aside>
      • <footer>
      • <form>
      • <video>

      Exemplo de box-level

      Parágrafos também são exemplos de box-level, mas os links são exemplos de inline-level. Vamos ver como tudo isso funciona

  • inline-level
    • Características
      • Sempre vai continuar na mesma linha do conteúdo, se tiver espaço
      • Exatamente do tamanho para conter o conteúdo da caixa
      • Não pula para a próxima linha, continua o conteúdo na mesma linha, se tiver espaço
    • Exemplos de tag's
      • <span>
      • <a>
      • <code>
      • <small>
      • <strong>
      • <em>
      • <sup>-<sub>
      • <label>
      • <button>
      • <input>
      • <select>
