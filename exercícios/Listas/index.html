Trabalhando com listas

Listas Ordenadas ou Listas OL - onde a ordem faz diferença

Usando a tag <ol> para criar uma lista ordenanda e colocando os itens da lista na tag <li> (obs: cada item em um <li> difrente)

  1. Acordar
  2. Arrumar a cama
  3. Tomar meu remédio
  4. Tomar banho
  5. Tomar café
  6. Escovar os dentes

Para mupo de lidar o tista basta adicionar o parâmetro type dentro de de <ol>

  1. Acordar
  2. Arrumar a cama
  3. Tomar meu remédio
  4. Tomar banho
  5. Tomar café
  6. Escovar os dentes

Caso queira que a lista comece em um número/algarismo/letra diferente de 1/I/a basta adicionar start ao lado de <ol type=""> - type é opcional

  1. Acordar
  2. Arrumar a cama
  3. Tomar meu remédio
  4. Tomar banho
  5. Tomar café
  6. Escovar os dentes

Listas UL ou Listas não Ordenadas

Usando a tag <ul> para criar uma lista não ordenada, onde obviamente a ordem dos itens não importa e colocando os itens da lista na tag <li> (obs: cada item em um <li> difrente)

  • Rap-10
  • Peito de peru
  • Queijo
  • Requeijão
  • Alface
  • Molho Rose
  • Azeite

Para mupo de lidar o tista basta adicionar o parâmetro type dentro de de <ul>
Há três tipos de tipos, tipo circle:

  • Rap-10
  • Peito de peru
  • Queijo
  • Requeijão
  • Alface
  • Molho Rose
  • Azeite

Tipo square:
  • Rap-10
  • Peito de peru
  • Queijo
  • Requeijão
  • Alface
  • Molho Rose
  • Azeite

E o tipo disc (padrão) que é o tipo da primeira lista não ordenanda.

Listas dentro de listas

Para adicionar uma lista dentro de outra basta apenas colcoar um <ol> ou um <ul> embaixo de um (ou mais de um, se for o caso de mais listas dentro de outras) componete da lista (<li>) principal

Meus exercícios favoritos:

  1. Costas:
    1. Remada pronada com o peito apoiado
    2. Puxada articulada
    3. Remada articulada
  2. Peito:
    • Supino inclinado
      1. Supino inclinado no Smith
      2. Supino inclinado com halter
      3. Supino inclinado na maquina
      4. Supino inclinado com barra
    • Supino reto
      1. Supino reto na maquina
      2. Supino reto no Smith
      3. Supino reto com halter
      4. Supino reto com barra
    • Voador
  3. Perna:
    • Agachamento
      1. Agachamento no Smith
      2. Agachamento livre
    • Stiff
    • Leg-Press
