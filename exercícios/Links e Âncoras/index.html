Usando links externos (HiperLink)

Você pode acessar o repositório público do Guanabara no GitHub - para colocar um hiperlink em alguma palavra ou frase basta utilizar a tag <a> e colocar o link de referência no href="(link de referência aqui)"

Abrir link em outra aba - Recomendado

Para fazer com que o link seja aberto em outra aba (sem fechar o seu site) basta adicionar o parâmetro target="_blank" rel="external" dentro da tag <a href="" aqui no caso>. Exemplo:

Links Internos

Fará a relação com outra página que está dentro do nosso servidor e diz respeito a nossa prórpia página. Exemplo:
Esta é a primeira página do site, se você quiser pode também acessar minha segunda página

Quando o link é interno não precisamos digitar a URL inteira. O link das demais páginas devem estar dentro da pasta do index

Colocar rel="next" (após <a href="nome do arquivo">) para páginas seguintes

Colocar rel="prev" (após <a href="nome do arquivo">) para páginas antecedentes

Você também pode acesar nossa página de notícias

Links para Download

Para baixar links após a tag <a href="nome do arquivo"> colocar download="arquivo" type=application/"tipo de arquivo"

  • Baixar arquivo em PDF - pdf está sendo reconhecido pelo chorme, por isso não está baixando
  • Arquivo compactado em ZIP
