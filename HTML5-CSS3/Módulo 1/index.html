Reproduzindo áudio

Vamos aprender a reproduzir áudios em HTML5

Parâmetros utilizados em <audio>

  • preload

    • Temos as seguintes configurações:

    • auto - onde o arquivo será carregado automaticamente
    • metadata - onde apenas a parte essêncial do arquivo é carregada
    • none - onde nada do arquivo é carregado até o usuário clicar
  • autoplay - para tocar automaticamente
  • controls - para aparecer os botões para tocar o arquivo/música

Videos - usar o site Pexels para baixar vídeos sem direitos autoriais

Este vídeo está hospedado no meu próprio servidor:

Formato m4v é o mais recomendado

Parâmetros usados em <video>: - nenhum é obrigatório e a maioria pode ser usado também para <audio>

  • width - para ajustar o tamanho do vídeo
  • poster - para adicionar uma imagem como capa do vídeo
  • controls - para aparecer os botões para iniciar o vídeo
  • loop - para o recomeçar assim que acabar
  • muted - para o vídeo não reproduzir som

Inserindo vídeo de outras plataformas - Recomendado para reduzir custos


Código:

<iframe width="tamanho do arquivo" height="peso do arquivo" src="URL do arquivo" title="título do arquivo" frameborder="tamanho desejado" allow=permições que você queira (como autoplay, picture-in-picture, web-share, entre outros) allowfullscreen></iframe>

Dica:

Como o youtube é do google, para os algoritimos darem maior abrangência para o seu site é recomendado subir o vídeo que você queira usar o no youtube e utilizar o método acima ↑

Para que utilizar esse?

Para reduzir custos quanto à hospedagem e trafego de dados, pois assim não vamos estar consumingo trafego de dados do nosso servidor, mas sim de um servidor de outra plataforma
