Testando o uso de iframe

  • Modo convencional de colocar link:
    • Acessando o site do CursoemVideo para aprender a programar
  • Usando o iframe:
    • Acessando o site do para aprender a programar
  • Caso o dispositivo não seja compatível com o iframe, podemos colocar dentro da tag <iframe> o link no qual o iframe está se referindo
  • IMPOTANTE:
    • Alguns sites como o Google tem uma proteção para que seu conteúdo não seja exibido em iframes
  • Por padrão todos os iframes tem tamanho de 300x150
    • Para alterar a altura e a largura basta mudar dentro da própria tag <iframe> os parâmetros "height" e "width"
    • Também podemos mudar esses parâmetros pelas CSS que serão soberanas às configurações anteriores
  • Podemos alterar a rolagem de conteúdo pelo parâmetro scolling que pode ter as seguintes configurações:
    • auto - automático
    • yes - mesmo que o conteúdo coubesse dentro do iframe a barra de rolagem apareceria
    • no - não vai aparecer a barra de rolagem
